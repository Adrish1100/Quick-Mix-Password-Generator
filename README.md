## 📖 Overview

Quick-Mix Password Generator is a collaborative beginner-friendly web project that generates **five unique and secure password variations** in a single click.

Users can choose their desired password length, and the application creates randomized passwords containing:

- Uppercase letters (A–Z)
- Lowercase letters (a–z)
- Numbers (0–9)
- Special symbols

The project emphasizes clean UI design, responsive layout, DOM manipulation, JavaScript loops, and secure random password generation. The landing page introduces the project and team before navigating to the password generator interface. 0

---

## ✨ Features

- 🔑 Generate **5 unique passwords** simultaneously
- 📏 Custom password length input
- ✅ Input validation with minimum and maximum limits
- 🔠 Uses uppercase, lowercase, numbers, and symbols
- 📋 One-click copy to clipboard
- 🌙 Modern dark-themed interface
- 📱 Responsive design
- ⚡ Built entirely using Vanilla JavaScript without external libraries

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/quick-mix-password-generator.git
```

2. Navigate to the project folder.

```bash
cd quick-mix-password-generator
```

3. Open `index.html` in your preferred web browser.

No additional dependencies or installation are required.

---

## ▶️ Usage

1. Open the landing page.
2. Click **Proceed to Generate**.
3. Enter the desired password length.
4. Press **Generate**.
5. Five secure passwords will be generated instantly.
6. Click the copy button beside any password to copy it to your clipboard.
